# 00_README_티_근거정리_작성원칙

본 문서는 티 추천 문구 작성 시 공통으로 지켜야 할 원칙을 정리하는 기준 문서입니다.

## 상태
- 이 파일은 기존에 0바이트(빈 파일) 상태였습니다.
- 현재 UTF-8 텍스트로 정상 읽기 가능한 기본 골격을 추가했습니다.

## 작성 원칙(초안)
- 건강효과를 단정적으로 표현하지 않습니다.
- 제품 레시피와 일치하지 않는 원료/클레임은 사용하지 않습니다.
- 결과 문구는 향미, 분위기, 시간대, 음용 상황 중심으로 작성합니다.
- 과장 없이 세련되고 따뜻한 톤을 유지합니다.
